# Laboratorio Cuántico: Experimento de Entrelazamiento (Bell State)

## Resumen
Este experimento simula un circuito cuántico de dos qubits diseñado para generar un estado de Bell, una forma máxima de entrelazamiento cuántico.

## Teoría
1. **Estado Inicial:** $|00\rangle$.
2. **Puerta Hadamard ($H$):** Aplicada a $q_0$.
3. **Puerta CNOT:** Entrelaza $q_0$ y $q_1$.

## Resultados
- **Vector de Estado:** `[0.5, 0.0, 0.0, 0.5]`
- **Interpretación:** El sistema colapsa solo en $|00\rangle$ y $|11\rangle$. Correlación perfecta.

## Conclusión
El Nodo GX1158 ha demostrado capacidad para procesar lógica cuántica de dos qubits.
